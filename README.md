# CabreraBrufau_et_al_2026_code
Code to reproduce the results of the article: Assessing the effects of restoration and conservation status on gaseous carbon fluxes and climate mitigation capacity across six European coastal wetlands.

Source data is published in LifeWatchERIC (https://doi.org/10.48372/C29B-QW38), and will openly available on January 1st 2028.
## Instructions:
First, place the source source data-files in the "0_sourcedata" folder.
Then simply run the scripts in order:
1. 1_Prepare_data.R
2. 2_Fit_GLMM_models.R
3. 3_Create_figures_and_tables.R

Folders will be automatically created by the scripts and they will be populated with figures and tables used in the article (preprint available at EarthArXiv with DOI: https://doi.org/10.31223/X5PB36).
